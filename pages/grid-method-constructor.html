grid.constructor(options)

Create a new instance of the Doby Grid object.

This will prepare the grid for rendering, do a lot of the data processing and screen calculations, but nothing will be rendered to the DOM until grid.appendTo() is called.

Parameters

options (object) - Grid configuration options

Options

Doby Grid provides a very large list of configurable options. They are documented separately under the "Grid Options" category on the left.

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid({
    columns: [{
        id: "id",
        name: "ID",
        field: "id"
    }, {
        id: "name",
        name: "Name",
        field: "name"
    }],
    data: [{
        id: 1,
        data: {
            id: 1,
            name: "John"
        }
    }, {
        id: 2,
        data: {
            id: 2,          
            name: "Steve"
        }
    }]
});
