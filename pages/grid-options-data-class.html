options.data.class

A custom CSS class to apply to this row. You can also pass in a function which returns a string.

Type and Default

(string), (function): Default undefined.

Example
// Simple class name definition
{class: "my-css-class"}

// Class definition via a function
{
    class: function (row, data) {
        return data.my_condition ? "Yes" : "No";
    }
}
