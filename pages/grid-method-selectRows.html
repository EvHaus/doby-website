grid.selectRows(fromRow, toRow, add)

Programmatically select a range of rows on the grid. The options.selectable parameter must be enabled for this to work.

Available Since

Version 0.0.8

Parameters

fromRow (integer) - Index of the row on which to start the selection.

endRow (integer) - Index of the row on which to end the selection.

add (boolean) - If true, will add selection as a new range, otherwise will replace existing selection ranges.

Returns

(object) - A Doby Grid class instance

Example
// Select the first row of cells
new DobyGrid(...).selectCells(0, 0);

// Select the first 5 rows
new DobyGrid(...).selectCells(0, 4);

// Select the first row, then add the second row to the selection
new DobyGrid(...).selectCells(0, 0);
new DobyGrid(...).selectCells(1, 1, true);
