grid.getState()

Retrieves a configuration object for the state of all user customizations for the grid. This allows you to easily restore states between page reloads.

You can listen to the statechange event to know when any user customizations have been done.

Parameters

options (object) - (Optional) Additional options for getting the state.

Options

column_properties (array) - By default, the grid will only store the 'id' and the 'width' properties of a column as those are the only two properties which are configurable from the front-end by default. However, in some cases, you may want to store additional column properties. This option allows you specify a list of additional properties to get from the state.

Returns

(object) - An object of values which should be passed into grid.restoreState().

Example
new DobyGrid(...).getState();

// Get the state of some additional column properties
new DobyGrid(...).getState({
	column_properties: ['name', 'my_custom_property']
});
