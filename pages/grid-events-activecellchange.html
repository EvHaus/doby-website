grid.events.activecellchange

Fired when the grid's active cell has been activated or changed to another cell.

Parameters

event (object) - Javascript event object

args (object) - This event's additional arguments.

Arguments

cell (integer) - The index of the cell the event is acting upon

item (object) - The data object for the specific row the event is acting upon.

row (integer) - The index of the row the event is acting upon.

Example
new DobyGrid().on('activecellchange', function(event, args) {
    //console.log(event, args)
})
