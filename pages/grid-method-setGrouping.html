grid.setGrouping(options)

Given a list of column grouping objects, will add grouping by those columns. The groups are assigned in the order of columns given.

Parameters

options (array) - A list of column grouping objects.

Options

A grouping object consists of the following key/value pairs:

options[x].class (string) - A custom CSS class name to add to this group's header row.

options[x].collapsed (boolean) - Should this grouping be collapsed? (Default true).

options[x].column_id (string) - ID of the column by which to group.

options[x].comparer (function) - Comparer function for sorting the grouping header rows.

options[x].formatter (function) - Formatter function for the grouping header row.

options[x].getter (function) - Getter function for pulling the value of the given column into the grouping header row.

options[x].groupNulls (boolean) - This option determines how null values should be handled. By default, null values will be grouped under a group called "null", however, you can set this option to false to hide the grouping row object for the "null" group.

options[x].rowSpacing (integer) - A custom row spacing to use for this group's header rows.

options[x].sticky (boolean) - If options.stickyGroupRows is used - should this grouping stick to the top? Default is true.

Returns

(object) - A Doby Grid class instance

Example
// Group by two columns
new DobyGrid(...).setGrouping([{column_id: 'first_category'}, {column_id: 'second_category'}]);

// Clear all groupings
new DobyGrid(...).setGrouping();
