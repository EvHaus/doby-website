grid.reset(models)

A way to update the data set for the grid in bulk, or to empty the grid.

Parameters

models (array) - (Optional) A list of data objects to set as the new data collection for the grid..

Returns

(object) - A Doby Grid class instance

Example
// Replace the existing data with a new data set
new DobyGrid(...).reset([{...}, {...}]);

// Empty the grid
new DobyGrid(...).reset();
