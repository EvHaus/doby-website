options.menuExtensions

This option allows you to add additional items to the grid's context menu. The option accepts a function which is expected to return an array of objects with the following supported attributes.

  • divider (boolean) - If true, this menu item will be a non-clickable divider line.
  • enabled (boolean) - If true, this menu item will be visible.
  • fn (function) - The function that will be executed when this menu item is clicked. More info below.
  • icon (string) - Optional icon to display at the front of the item. Can be an img tag or a custom bit of HTML code.
  • menu (array) - An array of submenu items.
  • name (string) - This is the text that will be displayed to the user for this menu item.
  • title (boolean) - If true, this menu item will be a non-clickable section title.
  • value (boolean) - If true, this menu item will display a checkmark icon.

The function you provide will be supplied with the following arguments:

  • event (object) - The Javascript event object from the click event.
  • grid (object) - The current instance of the Doby Grid.
  • args (object) - Various information about the area in which the user initiated the menu (with row, cell and column information).
Type and Default

(function): Default null.

Example
{
    menuExtensions: function (event, grid, args) {
        return [{
            icon: '',
            name: "Menu Item",
            menu: [{
                name: "Submenu Item",
                fn: function (event) {
                    // Perform some action here
                }
            }]
        }, {
            divider: true
        }, {
            name: "Toggle Button",
            value: your_option,
            fn: function (event) {
                your_option = !your_option;
            }
        }]
    }
}
