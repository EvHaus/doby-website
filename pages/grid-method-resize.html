grid.resize(models)

Re-size and redraw the grid using the current measurements of its container and viewport.

This will need to be called if you insert your grid into a hidden or off-screen DOM element. You do not need to call this when the browser's window is re-sized -- Doby Grid already automatically detects that.

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid(...).resize();
