grid.sortBy(column_id, ascending)

Quick method to sort the grid by a given column id.

Parameters

column_id (string) - The 'id' value of the column by which to sort.

ascending (boolean) - If 'true' the sort direction with be ascending, 'false' makes it descending.

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid(...).sortBy('myfirstcolumn', true);
