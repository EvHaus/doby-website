grid.events.columnchange

Fired when the columns have been changed.

Parameters

event (object) - Javascript event object

args (object) - This event's additional arguments.

Arguments

columns (array) - The current Column Options value.

oldColumns (array) - The Column Options before the change.

Example
new DobyGrid().on('columnchange', function(event, args) {
    //console.log(event, args)
})
