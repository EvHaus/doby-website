grid.removeColumn(column)

Removes a column from the grid.

Parameters

column (object), (string) - A column definition object, or a column 'id' value.

Returns

(object) - A Doby Grid class instance

Example
// Remove a column using the column definition object
new DobyGrid(...).removeColumn({id: 'mycolumn', name: 'My Column'});

// Remove a column using the column's id
new DobyGrid(...).removeColumn('mycolumn');
