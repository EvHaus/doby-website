options.tooltipType

The option handles the rendering style of tooltips as defined in the options.columns.tooltip option.

Your choices are:

  • title - For a default "title" attribute on the DOM element.
  • popup - For a custom popup using jQuery.

Note: This option cannot be toggled via grid.setOptions(). It must be set at initialization.

Type and Default

(string): Default "popup".

Example
{tooltipType: "title"}
