grid.selection

The selection object is made available to you for various selection-based commands. The grid's selection is stored as an array of Range objects which have a variety of helpful functions.

Type and Default

(array): Default null.

Example
// Select cells (0,0) to (5,5)
var grid = new DobyGrid(...).selectCells(0, 0, 5, 5);

// Get a list of all selected items for the first selection range
grid.selection[0].toRows();
