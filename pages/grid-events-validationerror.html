grid.events.validationerror

Fired when the cell editor's validation has failed.

Parameters

event (object) - Javascript event object

args (object) - This event's additional arguments.

Arguments

cell (integer) - The index of the cell the event is acting upon

cellNode (DOM Object) - The DOM for the cell the event is acting upon.

column (object) - The current options for the column that the event is acting upon.

editor (object) - The current Editor object.

row (integer) - The index of the row the event is acting upon.

validationResults (object) - The results of the editor's validation method.

Example
new DobyGrid().on('validationerror', function(event, args) {
    //console.log(event, args)
})
