grid.events.viewportchanged

Fired when the visible grid viewport is changed, like in the event of a scroll or page change.

Parameters

event (object) - Javascript event object

args (object) - This event's additional arguments.

Arguments

scrollLeft (integer) - The amount of pixels scrolled horizontally.

scrollTop (integer) - The amount of pixels scrolled vertically.

Example
new DobyGrid().on('viewportchanged', function(event, args) {
    //console.log(event, args)
})
