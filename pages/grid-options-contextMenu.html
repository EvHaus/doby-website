options.contextMenu

If enabled, will display a custom context menu for grouping, sorting, and editing other column settings. This option determines where the context menu will be toggled. Your choices are:

  • all - The context menu will appear when anything on the grid is right-clicked
  • header - The context menu will appear when the grid header is right-clicked
  • false - The context menu will be disabled

Note: This value cannot be toggled via grid.setOptions(). It must be set at initialization time.

Type and Default

(string), (boolean): Default all.

Example
{contextMenu: 'header'}
