What is doby.js?

doby.js is a web application framework created to help developers build beautiful web applications.

The framework is currenly in closed beta and not fully available to the public yet. We will be releasing the framework one module at a time for our early adopters who wish to test drive the features and provide us with feedback. We are also actively looking for web developers who wish to help contribute to our efforts.
