options.autoDestroy

This option controls whether or not the grid should cleanly destroy itself when it detects that the grid has been removed from the DOM. It adds a little extra overhead, but ensures that memory is freed up automatically.

With this option enabled you will not be able to insert the grid into an invisible or hidden element.

If you disable this option, make sure to manually call grid.destroy() when you want to remove the grid.

Type and Default

(boolean): Default true.

Example
{autoDestroy: false}
