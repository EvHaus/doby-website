grid.appendTo(target)

Similar to jQuery's appendTo(), will insert the grid element into the DOM at the end of the target.

It's only at this point that the grid becomes fully initialized, events get assigned and viewports calculated. This allows you to initialize your grid objects before inserting them into the DOM.

NOTE: It's important that the target element which you're inserting the grid into has a fixed height and width defined. The grid itself has a height of 0, and will rely on this target container to determine its dimensions.

Parameters

target (object) - A jQuery selector, DOM element, HTML string, or jQuery object into which the grid is inserted.

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid(...).appendTo($(document.body));
new DobyGrid(...).appendTo($('<div></div>'));
new DobyGrid(...).appendTo('#myelement');
