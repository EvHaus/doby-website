grid.showQuickFilter(column_id)

Allows you to programatically show and hide the Quick Filter bar below the column header.

Available Since

Version 0.0.8

Parameters

column_id (string) - ID of the column to focus after the Quick Filter is shown. If null, will hide the Quick Filter instead.

Returns

(object) - A Doby Grid class instance

Example

// Show the Quick Filter and focus the 'id' column input
new DobyGrid(...).showQuickFilter('id');

// Hide the Quick Filter
new DobyGrid(...).showQuickFilter();
