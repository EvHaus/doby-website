grid.selectCells(startRow, startCell, endRow, endCell)

Programmatically select a range of cells on the grid. The options.selectable parameter must be enabled for this to work.

If you don't pass it any parameters, all current selections will be deselected.

Parameters

startRow (integer) - Index of the row on which to start the selection.

startCell (integer) - Index of the cell on which to start the selection.

endRow (integer) - Index of the row on which to end the selection.

endCell (integer) - Index of the cell on which to end the selection.

Returns

(object) - A Doby Grid class instance

Example
// Select cells (0,0) to (5,5)
new DobyGrid(...).selectCells(0, 0, 5, 5);

// Deselect everything
new DobyGrid(...).selectCells();
