grid.resizeColumnsToContent(useMaxWidth, once, silent)

When resizeColumnsToContent is called, the grid will scan the currently rendered DOM elements to determine the maximum and minimum widths of the content of each cell and then resize the columns of the grid to ensure that all content fits.

WARNING: This is a very slow operation, especially on large grids and should not be called frequently as it will block rendering.

Available Since

Version 0.1.0

Parameters

useMaxWidth (boolean) - Take the maximum width of the columns into account

once (boolean) - Submit the column resize only once for all columns

silent (boolean) - Don't trigger resize events

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid(...).resizeColumnsToContent();
