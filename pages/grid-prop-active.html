grid.active

Detailed data about the currently active cell.

Type and Default

(object): Default null.

Example
// Active the first cell
var grid = new DobyGrid(...).activate(0, 0);

// View the active item
console.log(grid.active);
Sample
{
	cell: 2,
	node: <div.doby-grid-cell.l2.r2.active.editable>,
	row: 2
}
