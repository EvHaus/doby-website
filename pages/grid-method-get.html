grid.get(obj)

Get a model from the grid collection, specified by an id or by passing in a model.

Parameters

obj (object), (integer) - Data model or id value for the model to get.

Returns

(object) - A data model object.

Example
// Get by id
new DobyGrid(...).get(5);

// Get by model reference
new DobyGrid(...).get({id: 23, name: 'bob', age: 75});
