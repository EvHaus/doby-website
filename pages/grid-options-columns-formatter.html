options.columns.formatter

Function to use for formatting the cells in this column. Overrides the defaults set in options.formatter.

For additional documentation and instructions see: options.formatter

Type and Default

(function): Default undefined.

Example
{
    formatter: function (row, cell, value, columnDef, data) {
        return "My Special Value";
    }
}
