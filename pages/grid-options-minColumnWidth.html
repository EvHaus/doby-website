options.minColumnWidth

Restricts the minimum width the grid's column based on a given condition. Your available options are:

  • "" - No global restriction
  • "headerContent" - The minimum width of the columns will be set to that width the content in the column's header (ie. the name of the column plus some additional spacing for the sorting indicator)
Available Since

Version 0.0.8

Type and Default

(string): Default "".

Example
{minColumnWidth: "headerContent"}
