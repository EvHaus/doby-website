grid.showOverlay(options)

Hides all the rows of the grid and displays a single overlay element instead.

Parameters

options (object) - An object of additional options for the overlay.

Options

class (string) - Custom CSS class to use for the overlay.

html (string) - Custom HTML to insert into the overlay.

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid(...).showOverlay({
    html: "There was an error loading the grid"
});
