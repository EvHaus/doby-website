grid.restoreState(state)

Restores the state of grid's user customizations. Expect an object which was returned from grid.getState().

If possible, you should call this method before calling grid.appendTo() to prevent unnecessary redraws.

You can listen to the statechange event to know when a user customization has been done.

Parameters

state (object) - The object of settings from grid.getState().

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid(...).restoreState({...}).appendTo(document.body);
