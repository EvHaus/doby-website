options.columns.class

An additional CSS class name to add to every cell in this column. You can also pass in a function which returns a string.

Type and Default

(string), (function): Default null.

Example
// Simple class name definition
{class: 'cell_in_my_column'}

// Class definition via a function
{
    class: function (row, cell, value, columnDef, data) {
        return data.my_condition ? "Yes" : "No";
    }
}
