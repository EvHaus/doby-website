grid.getRowFromEvent(event)

This helper function allows you to fetch a given row model from a Javascript event object. This is useful when creating custom events outside the grid's event system.

Parameters

event (object) - Javascript event object.

Returns

(object) - A data model object.

Example
var grid = new DobyGrid(...);

$my_row.on('click', function (event) {
    var row = grid.getRowByEvent(event);
});
