options.columns.exporter

Similar to the options.columns.formatter option, this function allows you to define a special way for data to be exported for this column during any export operations (ie. grid.export(), or when cells are selected and sent to the clipboard).

This is useful when you have an image or another graphical value inside you cell, but still want some meaningful value to be exported back to the user that isn't the default data field.

Type and Default

(function): Default undefined.

Example
{
    exporter: function (columnDef, data) {
        return "My Special Value";
    }
}
