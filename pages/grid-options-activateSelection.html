options.activateSelection

When using click'n'drag to select a range of cells, this option determines whether or not the active cell should be set during this operation. If disabled, click'n'drag only selects a range of cells. If enabled, cells are still selected and the last cell in the range is automatically activated for edit allowing you to take an action.

Type and Default

(boolean): Default true.

Example
{activateSelection: true}
