grid.remoteFetchAll(callback)

In some rare cases, you way need to have your grid fetch all available rows for a collection. This methods allows you to do this.

Remote Data must be enabled for this to come into effect.

Parameters

callback (function) - (Optional) Callback function to execute after the data is fetched.

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid(...).remoteFetchAll();

new DobyGrid(...).remoteFetchAll(function () {
	// After results are fetched
});
