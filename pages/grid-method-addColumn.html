grid.addColumn(data, options)

Add a new column definition to an existing grid instance.

Parameters

data (object) - Column data options

options (object) - (Optional) Additional options for handling the insert.

Options

at (integer) - The index at which to splice the column into the grid.

merge (boolean) - Check to see if columns with the given id already exist. If they do - merge their data.

Returns

(object) - A Doby Grid class instance

Example
// Add a column at the front of the grid
new DobyGrid(...).addColumn({...}, {at: 0})

// Add a column at the end of the grid
new DobyGrid(...).addColumn({...})
