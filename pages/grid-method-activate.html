grid.activate(row, cell)

Activate a cell on the grid. Calling this function with no arguments will deselect any active cells.

Parameters

row (integer) - The index of the row in which your cell will be activated.

cell (integer) - The index of the cell in the row you want to activate.

Returns

(object) - A Doby Grid class instance

Example
// Activate the first cell
new DobyGrid(...).activate(0, 0)

// De-activate all cells
new DobyGrid(...).activate()
