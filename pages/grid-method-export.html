grid.export()

Export all grid data to a format of your choice.

The results will be returned to you via a callback function.

Parameters

format (string) - Which format to export to. Available formats are:

  • csv - For comma separate value string
  • html - For an HTML table of data

callback (function) - A callback function which will return the exported data.

Returns

(object) - A Doby Grid class instance

Example
// Export the grid to a CSV string
new DobyGrid(...).export('csv', function (result) {
    // Process the 'result' here
});

// Export the grid to an HTML string
new DobyGrid(...).export('html', function (result) {
    // Process the 'result' here
});
