grid.setSorting(options)

Sets the sorting for the grid.

Parameters

options (array) - List of column sorting options to use.

Options

Each column sorting options object should have two parameters:

options[x].columnId (string) - The 'id' value of the column by which to sort.

options[x].sortAsc (boolean) - If true the sort direction with be ascending, false makes it descending.

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid(...).setSorting([
    {columnId: 'myfirstcolumn', sortAsc: true},
    {columnId: 'myothercolumn', sortAsc: false}
])
