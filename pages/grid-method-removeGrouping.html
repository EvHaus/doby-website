grid.removeGrouping(column)

Ungroups the grid for a given column grouping.

Parameters

column (object), (string) - A column definition object, or a column 'id' value.

Returns

(object) - A Doby Grid class instance

Example
// Remove a column's grouping using the column definition object
new DobyGrid(...).removeGrouping({id: 'mycolumn', name: 'My Column'});

// Remove a column's grouping using the column's id
new DobyGrid(...).removeGrouping('mycolumn');
