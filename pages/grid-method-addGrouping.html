grid.addGrouping(data, options)

If options.groupable is enabled, this will add a new grouping layer to the grid. It allows you create nested groups.

You can pass in additional options to override default grouping settings.

Parameters

column_id (string) - Id of the column by which to group by.

options (object) - (Optional) Custom grouping options for this group.

Options

See grid.setGrouping() for information about grouping options.

Returns

(object) - A Doby Grid class instance

Example
// Group data by the values in the 2nd column
new DobyGrid(...).addGrouping('my_second_column');

// Group data by the values in the 2nd column, but automatically expand the group
new DobyGrid(...).addGrouping('my_second_column', {
    collapsed: false
});
