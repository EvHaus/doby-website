options.resizeCells

If enabled, when resizing columns via mouse drag and drop, will resize the cells of all column rows. Otherwise will only resize the headers until you stop dragging. The options.resizableColumns option must be enabled for this to work.

This option is disabled by default because on large grids, the resizing of cells is choppy and slow. If you're working with smaller grids - you may want to enable this for a better user experience.

Type and Default

(boolean): Default false.

Example
{resizeCells: true}
