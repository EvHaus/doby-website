options.forceRemoteSort

By default, when using a Remote Data, DobyGrid will switch to local (client-side) sorting once the full data set has been fetched. In some cases this is undesireable as you may want to have your server perform all sorting operations. This option allows you to force DobyGrid to always refetch data when performing a sort.

Available Since

Version 0.1.0

Type and Default

(boolean): Default false.

Example
{forceRemoteSort: true}
