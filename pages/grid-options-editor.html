options.editor

When editing cells, this will be the custom editor function to use as the default. The options.editable option must be enabled for this to go into affect. If a specific column has another editor specified via the "Column Options" section on the left, this option will be ignored for that column. For additional documentation on editors see: Editors.

Type and Default

(function): Default false.

Examples
{
    editor: function (options) {
        ...
    }
}
