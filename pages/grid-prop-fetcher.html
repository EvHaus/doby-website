grid.fetcher

The remote data fetching object instance that is created when using a Remote Data collection.

The grid.fetcher.request object is especially useful if you want to manually control fetching after the grid makes requests (for example, to abort a long-running request).

Type and Default

(object): Default {}.

Example
var grid = new DobyGrid(...);

// View the current fetcher
console.log(grid.fetcher);
Sample
{
	count: function (options, callback) {},
	fetch: function (options, callback) {}
	fetchGroups: function (options, callback) {},
	grid: DobyGrid
	onLoaded: function () {},
	onLoading: function () {},
	
	// Reference to the currently active AJAX request object
	request: {}
}
