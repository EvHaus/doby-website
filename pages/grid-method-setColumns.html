grid.setColumns(columns)

Given a new column definition array - will update the grid to use the new definitions. Allows you to perform bulk grid.addColumn()/grid.removeColumn() operations.

Parameters

columns (array) - A list of column definition objects, as per Column Options

Returns

(object) - A Doby Grid class instance

Example
new DobyGrid(...).setColumns([
    {id: 'mycolumn', name: 'My Column'},
    {id: 'anothercolumn', name: 'My Other Column'}
]);
