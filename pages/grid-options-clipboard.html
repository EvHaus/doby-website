options.clipboard

This option determines how the grid data will be sent to the clipboard when cells are selected and Ctrl/Command + C is pressed. Your available options are:

  • csv - Data will be stored as a comma separated value string
  • json - Data will be stored as a JSON array
  • html - Data will be stored as an HTML table element
Type and Default

(string): Default csv.

Example
{clipboard: 'json'}
Demo

Press "Select All", or click and drag inside the grid to select a range of cells. Then press Ctrl/Command + C to copy the selection. Then Paste the selection into a text editor of your choice using Ctrl/Command + V.
