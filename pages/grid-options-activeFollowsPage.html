options.activeFollowsPage

When using keyboard shortcuts (Page Up and Page Down) to scroll up and down the grid, the default behaviour is to keep the activated cell unchanged. If you enable this option, the activated cell will change with each page move. ie. If you select the top right cell and then page down, the new active cell will be the top right cell of the next page.

Type and Default

(boolean): Default false.

Example
{activeFollowsPage: true}
