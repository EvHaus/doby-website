options.lineHeightOffset

Depending on the font you wish to use, sometimes you need to offset the line-height styling by one or two pixels to ensure the text is positioned nicely in the middle of each cell. Use this value to control that offset in pixels.

This option does not affect the grid header. The header's height and line-height can be customized via CSS.

Type and Default

(integer): Default -1.

Example
{lineHeightOffset: 0}
