grid.sorting

The current column sorting that is applied on the grid instance.

Type and Default

(array): Default {}.

Example
var grid = new DobyGrid(...);

// View the current sorting
console.log(grid.sorting);
Sample
[{
	columnId: "column-a",
	sortAsc: true
}, {
	columnId: "column-b",
	sortAsc: false
}]
