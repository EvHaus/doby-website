options.dataExtractor

A custom function to use for extracting data item object values from your options.data object. This is useful if you want to use a custom options.data object format that Doby Grid doesn't recognize by default.

The function takes in two arguments: item and columnDef, where the item is a data object for the given row and columnDef is the column definition object for the column requested. See example for more clarity.

Type and Default

(function): Default null.

Examples
{
    data: [{
        mydata: {
            id: 1,
            name: "Magical Item #1"
        }
    }, {
        mydata: {
            id: 2,
            name: "Magical Item #2"
        }
    }]
    dataExtractor: function (item, columnDef) {
        // For group headers, simply return the value by which the group was groupped
        if (item.__group) return item.value;

        // For regular data items, the object is inside a special 'mydata' key.
        return item.mydata[columnDef.field];
    }
}
