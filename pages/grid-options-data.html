options.data

This option holds the main data set or method for fetching a remote data set. The data can be either:

  • an array of objects
  • a Backbone.Collection object
  • a function method for remote data fetching

If you wish to use a simple list of data objects, see the "Data Options" section on the left for more info.

For more information on using remote data sources see Remote Data.

Type and Default

(array), (object), (function): Default [].

Examples
// Simple data array
{
    data: [{
        data: {
            id: 1,
            name: "Magical Item #1"
        }
    }, {
        data: {
            id: 2,
            name: "Magical Item #2"
        }
    }]
}
// Remote data fetching function
{
    data: function() {
        // See "Remote Data" page for more info
    }
}
// Backbone Collection data set
{
    data: new Backbone.Collection()
}
