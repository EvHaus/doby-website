options.formatter

Default function to use for all cell formatting. If a specific column has another formatter specified via the "Column Options" on the left, this option will be ignored.

The formatter takes in the following arguments:

row (integer) - Index of the row when the cell is located.

cell (integer) - Index of the cell in the specified row.

value (string) - The value of this cell from the data object for this row.

columnDef (object) - The column definition object for the given column.

data (object) - The full data object for the given cell.

The formatter function will be bound to the grid instance, so you can reference the grid via this.

Type and Default

(function): Default null.

Examples
// Simple formatter that renders the value of the cell directly
{
    formatter: function (row, cell, value, columnDef, data) {
        return value;
    }
}

// Cleaner formatter which strips out HTML characters and handles
// empty values nicely (this is similar to what Doby Grid will do by default
// if you don't specify any formatter at all)
{
    formatter: function (row, cell, value, columnDef, data) {
        if (value === undefined || value === null) {
            return "";
        }

        return (value + "")
            .replace(/&/g, "&amp;")
            .replace(/</g, "&lt;")
            .replace(/>/g, "&gt;");
    }
}
