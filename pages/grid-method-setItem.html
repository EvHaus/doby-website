grid.setItem(item_id, attributes)

Replaces an item in the grid with new data.

If your grid is using a Backbone.Collection as your data set, the attributes parameter must be a valid Backbone.Model object.

Parameters

item_id (string) - The 'id' of the item you wish to update.

attributes (object) - A full data item to replace the existing item with, or a Backbone.Model object.

Returns

(object) - A Doby Grid class instance

Example
// Updating a simple data set
new DobyGrid({
    data: {
        id: 123,
        data: {
            name: 'bob'
        }
    }
}).setItem(123, {
    id: 123,
    data: {
        name: 'steve'
    }
});

// Updating a Backbone.Collection data set
new DobyGrid({
    data: new Backbone.Collection({id: 1, name: 'bob'})
}).setItem(123, new Model({name: 'steve'});
