options.columns

List of column definition object to use for rendering the grid. See "Column Options" section for more info about available options.

Type and Default

(array): Default [].

Example
{
    columns: [{
        id: "myfirstcolumn",
        field: "id",
        name: "ID"
    }, {
        id: "mysecondcolumn",
        field: "foobar",
        name: "Foobar"
    }]
}
