options.scrollLoader

A function that will return an HTML string to display during virtual scrolling. By default, nothing will be displayed - which may be undesired. This allows you to display a loading spinner or message.

Type and Default

(function): Default null.

Example
{
    scrollLoader: function () {
        return "Loading...";
    }
}
Demo

Use the scrollbar to quickly scroll up and down to see the scrollLoader message.
