grid.add(models, options)

Add a model (or an array of models) to the grid collection. Pass {at: index} as an option to splice the model into the collection at the specified index. If you're adding models to the collection that are already in the collection, they'll be ignored, unless you pass {merge: true}, in which case their attributes will be merged into the corresponding models.

Parameters

models (object), (array) - Data model(s) to insert

options (object) - (Optional) Additional options for handling the insert.

Options

at (integer) - The index at which to splice the model(s) into the collection.

forced (boolean) - If enabled, will force a grid refresh regardless of how frequently add is called. Otherwise, will debounce it for 10ms. Default is false.

merge (boolean) - Check to see if models with the given id already exist. If they do - merge their data.

silent (boolean) - If true, will not auto-reload the grid - only add the items to the internal collection.

Returns

(object) - A Doby Grid class instance

Example
// Insert a row at position 5
new DobyGrid(...).add({...}, {at: 4})

// Insert two new rows to the end of the grid
new DobyGrid(...).add([{...}, {...}])
